Playwright Practice

This repo holds my practice code while learning Playwright.
I use it to test ideas and try different ways to handle UI checks.
The project helped me get used to writing stable tests and organizing files in a clear way.

The code shows how I handle locators, selectors, fixtures, waits, reports and cross browser runs.
It is a mix of small examples and small test flows that build on what I learned step by step.

This repo is not a polished project.
It is more like a record of my progress while getting used to Playwright.
Anyone who checks it out can see how I write tests and how I structure basic flows.

If someone wants to run it they can install Playwright with the usual setup and run the tests with the standard command.
Everything else should work as long as the environment has Node and the Playwright package.
